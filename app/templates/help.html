Guidelines for using this project

  • Requirements

    1. 1. Pythonn 2.7 or above
    2. 2. Flask Framework
    3. 3. Cassandra Python Driver


  • Nodes

    1. 1. Nodes are the cassandra database servers.
    2. 2. you can add a new node by providing IP address.
    3. 3. By default it uses localhost as cassandra database server.


  • Keyspaces

    1. 1. Keyspace are the container for your application data, similar to a schema in a relational database.
    2. 2. This part of project will fetch all keyspaces available at given cluster node of cassandra database.
    3. 3. On clicking on particular keyspace it will lead you to column families in that keyspace.
    4. 4. By clicking on particluar column family, this tool will fetch all rows from that column family.


  • Column Families

    1. 1. In this section, all column families fom all the keyspaces which are present at given cluster node are fetched.
    2. 2. If you are interested in column family of particular keyspace then go to keyspace tab.
    3. 3. On clicking on column family name it will redirect to page that provide schema of that column family.
    4. 4. Here you can choose which attribute you want to fetch and then submit the query. The tool will fetch you all rows with given columns


    To know more about Cassandra Database please visit Apache Cassandra
